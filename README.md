# Clinic Appointment Management System

A full-stack web application for managing clinic appointments across multiple medical departments.

## Tech Stack
- **Backend:** Python (Flask)
- **Database:** MySQL
- **Frontend:** HTML5, CSS3, JavaScript

## Features
- Book appointments by department
- View, update, and delete appointments
- Real-time database integration
- JavaScript form validation

## How to Run
1. Clone the repo
2. Install dependencies: `pip install flask mysql-connector-python`
3. Import `sql_web.sql` into MySQL
4. Run `python app.py`
5. Open `http://localhost:5000`
